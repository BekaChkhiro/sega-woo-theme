# T9.6: New Products Carousel Section

## Overview
Implemented the "New Products" carousel section on the homepage using the ProductCarousel Blade component.

## Implementation Details

### Location
- **File**: `resources/views/front-page.blade.php` (lines 102-129)

### Features
- Displays up to 12 newest products in a responsive carousel
- Touch-friendly swipe navigation on mobile devices
- Navigation arrows on desktop (hidden on mobile)
- Smooth animations and transitions
- "View All" link to shop page filtered by date

### Configuration
```blade
<x-product-carousel
  :products="$newProducts"
  :title="__('New Arrivals', 'sage')"
  :slides-per-view="4"
  :space-between="24"
  :autoplay="false"
  :loop="true"
  :navigation="true"
  :pagination="false"
  :view-all-url="wc_get_page_permalink('shop') . '?orderby=date'"
/>
```

### Query
```php
$newProducts = wc_get_products([
  'limit' => 12,
  'orderby' => 'date',
  'order' => 'DESC',
  'status' => 'publish',
  'visibility' => 'visible',
]);
```

### Responsive Behavior
- **Mobile** (< 480px): 1 product per slide
- **Small Mobile** (>= 480px): 2 products per slide
- **Tablet** (>= 768px): 3 products per slide
- **Desktop** (>= 1024px): 4 products per slide
- **Large Desktop** (>= 1280px): 4 products per slide
